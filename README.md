# TaskMild 公益版

## 简介

TaskMild 是一个以稳定、低功耗、低内存占用为核心目标的进程压制模块。

它不是简单粗暴地一刀切杀后台，而是以进程为单位进行更细粒度的控制，只对进入候选范围的目标进程进行判断和处理，同时默认避开前台进程、主进程以及关键进程，尽量在不影响正常使用的前提下减少不必要的后台活动。

公益版的目标不是堆功能，而是把最常用、最必要的能力做得更简单、更稳、更适合长期使用。

---

## 设计目标

- 稳定优先
- 低功耗优先
- 低内存占用
- 进程级控制
- 配置尽量简单
- 白名单优先
- 默认保护前台与关键进程

---

## 功能特性

- 进程级压制，而不是整包一刀切
- 白名单优先
- 默认前台保护
- 默认主进程保护
- 默认关键进程保护
- 支持熄屏场景
- 支持后台场景
- 支持三档预设模式
- 单主程序结构
- 单配置文件
- 支持基础调试和状态查看
